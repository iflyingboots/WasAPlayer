NeteasePlayer
=============

A simple command line music player tailored for NetEase Music.

##Screenshot
![](http://incognita.u.qiniudn.com/NeteasePlayer_screenshot_v0.2.png)

##Details
Written in `Node.js`

##Usage
###Required components `npm install ...`
* request
* colorful
* prompt
* player
* term-list-bar

### Run
`$./bin/cli`
